Welcome to the Internet Clipboard

This application allows you to save and retrieve text content using custom URLs.

To save content:

  • Send a POST request to /api/<your-custom-url> with the content in the request body.

To retrieve content:

  • Send a GET request to /api/<your-custom-url> to get and delete the content.

Alternatively, you can use the form available at /<your-custom-url> to save or retrieve content via the browser.
